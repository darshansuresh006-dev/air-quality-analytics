# Air Quality Analytics and Insight Report

Project 01 — Data Analysis Foundation

## Problem
Urban air-quality sensor data is noisy, incomplete, and hard to interpret directly.
This project cleans raw hourly sensor readings and produces evidence-based insights
about pollution patterns and data quality.
